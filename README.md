# 0.前言
Python是一门解释型、面向对象的、带有动态语义的高级程序语言。它有许多发行版本，例如ActivePython、StacklessPython，这两个版本都是标准的Python实现由c语言编写。除此还有Jython和TronPython等等这些是由其他语言实现。
